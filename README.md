# HoopDraft 🏀

NBA Draft trivia game. Guess the pick, first NBA team, college, and year for 150+ current NBA players.

## Tech stack

- React 18 + Vite
- Tailwind CSS
- Lucide icons

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Deploy to Vercel

See the full walkthrough in your chat — the short version:

1. Push this folder to a new GitHub repo
2. Go to [vercel.com/new](https://vercel.com/new) and import it
3. Click Deploy — done in ~60 seconds
